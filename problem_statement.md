## Problem Statement

- Traffic congestion in urban areas causes significant delays, fuel wastage, and environmental pollution, affecting both developed and developing cities globally.

- Existing VANET-based congestion detection systems, including Mohanty et al. (2025), rely solely on simulation environments like SUMO and classical fuzzy methods (FKM + FAHP) which limits their detection accuracy and real-world applicability.

- Current research produces results only as static tables and numbers in MATLAB with no visual interface, making findings inaccessible to actual end users like traffic authorities and drivers.

- No existing system provides an end-to-end pipeline that connects congestion detection directly to real-time user notification and alternate route suggestion in a single unified platform.

- Individual sensor failures, communication breakdowns in dense traffic, and lack of adaptive learning in static fuzzy models further reduce the reliability of current approaches.

- There is no deployable prototype that translates VANET-based congestion research into a usable product for real-world traffic management scenarios.

- This project addresses these gaps by proposing an improved machine learning based congestion detection system integrated with a real-time web dashboard for traffic authorities and a mobile application for drivers, delivering a complete, user-facing smart urban mobility prototype built on SUMO simulation data.
